# birdie69-docs

> Project documentation hub for **birdie69** — a daily-question relationship app for couples.

---

## What is birdie69?

birdie69 gives couples one question per day. Both partners answer independently, then the answers are revealed simultaneously — sparking honest conversations and deeper connection.

---

## This Repository

`birdie69-docs` is the **single source of truth** for all project documentation:

| Directory | Contents |
|-----------|---------|
| `adrs/` | Architecture Decision Records (ADR-001 to ADR-005+) |
| `architecture/` | System context, component diagrams, sequence diagrams |
| `requirements/` | Product requirements, user stories |
| `planning/` | Backlog, sprint plans |
| `api-specs/` | OpenAPI 3.1 specifications |
| `.claude/` | AI agent prompts and rules |
| `scripts/` | Confluence sync script |
